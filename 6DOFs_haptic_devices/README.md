# 6DOFs_haptic_devices

Drivers and libraries for the N70 and N71 haption haptic devices 

## INSTALL

Follow the instruction in tutorial.txt. The ip are already modified in order to run the two device simultaneously.

n70 IP: 192.168.9.16

n71 IP: 192.168.9.17

Don't forget to set the local pc ip to 192.168.9.101 (gateway 255.255.255.0)

## ROS1 wrapper

	https://downloads.haption.com/projets/ROS/ROS-Virtuose_20220617.zip
	
## ROS2 wrapper 

	https://github.com/idra-lab/haptic_interface_ROS2/tree/main
	
In addiction copy in /etc/Haption/Connector the file desktop_6D_nXX.param
